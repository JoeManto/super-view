You can find all the things SuperWindow can do via the menu bar icon. Simply right or left click our icon to be presented with these operations.

Opening a Web Window vs Opening a local file

When you get to opening up windows you may notice that there are two different options Open Web Window and Open Local File. Use the local file operation when wanting to use a pdf reference etc. Use the Web Window when accessing content or resources off your computer.

Moving Windows

When clicking on a move operation this move will be applied to the current SuperWindow in use

Requesting Features

Please find the Request Feature operation under the support menu tab. This will give you the required contact details and recommendations on your feature requests

Happen to find a bug?

Please find the Report a Bug operation under the support menu tab. Here will give you all the details we need to make sure your found bug gets fixed.
